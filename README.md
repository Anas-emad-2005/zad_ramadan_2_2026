# زاد رمضان

تطبيق رمضاني يومي للتعبد والأذكار وتتبع العبادات — صُنع بحبٍّ وإخلاص.

---

## هيكل المشروع

```
zad-ramadan/
├── index.html
├── README.md
└── assets/
    ├── css/
    │   └── styles.css          ← نظام التصميم الكامل
    ├── js/
    │   ├── storage.js          ← إدارة LocalStorage
    │   ├── content.js          ← الآيات والأذكار اليومية
    │   ├── adhkar.js           ← أذكار الصباح والمساء والنوم
    │   ├── rewards.js          ← بيانات الجوائز ومنطق الفتح

    ├── images/                 ← (اختياري)
    └── audio/
        └── click.mp3           ← صوت النقر (اختياري)
```

## الميزات

| الميزة | التفاصيل |
|--------|----------|
| **المحتوى اليومي** | آية قرآنية + ذكر + تذكير، تتغير بحسب اليوم |
| **الأذكار** | أذكار الصباح / المساء / النوم في accordion |
| **أفكار الصدقة** | قائمة صدقات جارية منظّمة |
| **العبادات اليومية** | 5 مهام بنظام رصيد صارم (مرة واحدة فقط في اليوم) |
| **نظام الرصيد** | تراكمي، لا يُعاد تعيينه، مع شريط تقدم ودائرة |
| **المستويات** | بداية طيبة / ماشي صح / ممتاز / واصل |
| **التسلسل اليومي** | يُحسب فقط عند إكمال ≥٣ مهام، ويُعاد تعيينه عند انقطاع يوم |
| **الجوائز** | تُفتح كل ٥٠ نقطة، تظهر داخل modal |
| **زر الإهداء** | رسالة إهداء الأجر لموتى المسلمين |

## مفاتيح LocalStorage

| المفتاح | الوصف |
|---------|-------|
| `zad_total_points` | إجمالي الرصيد التراكمي — **لا يُعاد تعيينه أبداً** |
| `zad_streak` | عدد الأيام المتواصلة |
| `zad_last_completed_date` | آخر يوم أُنجز فيه ≥٣ مهام |
| `zad_daily_tasks_YYYY-MM-DD` | المهام المُنجزة لهذا اليوم — **تُعاد كل يوم** |
| `zad_unlocked_rewards` | الجوائز المفتوحة — **لا تُعاد أبداً** |

---

## القواعد المنطقية لنقاط مهمة

- **مهمة واحدة في اليوم**: تُحفظ في `zad_daily_tasks_YYYY-MM-DD`. إعادة التحميل لا تُضيف نقاطاً.
- **التسلسل**: يزيد فقط إذا كان آخر إنجاز بالأمس تحديداً. لا يزيد عند إعادة التحميل في نفس اليوم.
- **يوم مفقود**: يُعاد التسلسل إلى 0 عند تحميل الصفحة إذا كان آخر إنجاز قبل أكثر من يوم.
